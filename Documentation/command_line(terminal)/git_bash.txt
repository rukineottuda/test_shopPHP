ls покажет список файлов
pwd покажет каталог в каком нахожусь
cd переход на уровень выше ниже
cd .. зайдём в папку
cd с:/ переход в определённую директорию
cd testfolder - вернуться где были
clear очистка консоли

mkdir testfolder создать папку
cd !$  переходит в  папку которая только что создалась
mkdir -p   app/{css,js} позволит создать несколько файлов
touch app/{css,js}  позволит создать несколько файлов
touch .bowerrc создаст скрытый файл bower
ls -f покажет скрытые файлы
rm  удалить  если папка не пуста не удалиться надо добавить флаг rm -r testfolder
