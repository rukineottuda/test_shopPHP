3)Определить связи между сущностями
Виды связей:
Один-ко-многим- одна запись в из одной таблицы может связана с множеством записей в другой таблице
Многие-ко-многим - множеству записей из одной таблицы могут соответсвовать множество записей из другой таблицы.
Один-к-одному- экземпляру одной сущности соответствует только один экземпляр другой.

Как опознать связь?
1)Сколько объектов B могут относиться к объекту А?
2)Сколько объектов A могут относиться к объекту B?

6)Определить типы полей
MySQL поддерживает несколько типов данных
Числовые данные-это целые числа (без дробной части) и вещественные(с дробной частью).
Строковые данные- последовательность символов заключённая в кавычки.
Календарные данные - тип для обозначения даты и времени, принимает различные формы, например
строковую "2007-11-23" или числовую 20071123
NULL-специальный тип данных, обозначает отсутствие информации.
